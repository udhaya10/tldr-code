Root cause:
  Rust raw-string literals like  r"(?m)^\\s*export\\s+default..."  compile to a
  regex containing literal `\s` (a backslash followed by `s`) — NOT the
  whitespace character class. The pattern therefore never matches real source
  containing actual whitespace.
